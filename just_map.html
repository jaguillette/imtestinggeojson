Historical Gazetteer of Russia Map

This map shows the locations of various market towns in Russia in the late 1700s

View project on GitHub

1st-level administrative units show up as stars, 2nd-level administrative units as squares, and individual towns as circles.

All markers are colored by 1st-level administrative division.

Only individual towns contain descriptions from source material. The descriptions also contain the url to access the original document on Hathi Trust.
